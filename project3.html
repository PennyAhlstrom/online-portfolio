C# .NET Project Management App

In our Web Application course, we undertook a comprehensive exploration of the
Model-View-Controller (MVC) architecture by developing a project management application
using .NET and C#. This series of sequential labs spanned the entire semester,
with each session building upon the previous one to gradually expand the functionality
of the application. Over the course of the semester, we focused on creating a robust,
scalable app, implementing key features such as task management, user authentication,
and project tracking. By following a structured approach, we developed an in-depth
understanding of the MVC pattern and its role in creating maintainable and efficient
web applications.

Technologies Used

.NET, C#, HTML, CSS, Postgres, SQL, git, GitHub
